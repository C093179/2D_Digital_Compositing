# 2D
2D - x,y 함수  / 3D - x,y,z (depth)

* 표현방식 = 2D, 3D 

* 최종작품을 완성하는 매체 = **only 2D**(아직 기술의 발달이 덜 됨)

     --> 2D는 단순한 이미지의 합성 차원이 아닌 결과물을 표현하기 위한 가장 기본적인 개념(3D 분야와 밀접한 관련)
     
         맵핑, 합성등을 할 때에도 2D의 방식이 필요하기도 함
